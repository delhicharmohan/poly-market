# Icon Generation Instructions

To generate PWA icons, you can use any image editor or online tool:

1. Create a square image (512x512px recommended)
2. Use the Indimarket logo or a simple icon design
3. Export as PNG files:
   - `icon-192.png` (192x192px)
   - `icon-512.png` (512x512px)

Alternatively, you can use online tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Place the generated icons in the `/public` directory.
